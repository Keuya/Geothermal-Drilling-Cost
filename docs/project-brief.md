# Project Brief

## Central question
What drives geothermal drilling cost, and how sensitive project economics are to drilling outcomes?

## Initial scope
- Explore cost drivers by depth, rig type, logistics, and non-productive time.
- Model uncertainty in successful well ratio.
- Connect drilling outcomes to levelized project economics.

## Geographic focus
Primary anchor market: Kenya.
Secondary comparisons: Iceland, Indonesia, Philippines (as data allows).

## Deliverables
- Reproducible dataset and assumptions log
- Baseline drilling risk model
- Scenario charts and summary tables
- Finance interpretation for bankability and de-risking tools
